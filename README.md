# Life Tracker (Expo + SQLite)

V1 iPhone-focused Life Tracker built with Expo Router and `expo-sqlite`.

## Included in V1

- Bottom tab navigation
- Today screen with minimum wins checklist
- Week screen with recurring anchors
- Habits screen with streak tracking + Grace Mode (2 grace uses/week)
- Journal screen with daily entries
- 3D Projects screen with project cards and print logs
- Settings screen

## Run

```bash
npm install
npm run start
```

Open with Expo Go on iOS.
